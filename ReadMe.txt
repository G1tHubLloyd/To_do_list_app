# 📝 To-Do List App

A lightweight and interactive to-do list built with jQuery and jQuery UI, designed for broad browser compatibility and ease of use. Add tasks, mark them complete, delete them, and reorder your list with intuitive controls.

---

## 🚀 Features

- ✅ Add tasks with a button click or by pressing Enter
- ✍️ Double-click items to mark as completed
- ❌ Delete tasks with a single click
- 🧩 Drag and drop to reorder tasks
- 🔒 Compatible with older browsers (no polyfills required)
- 🎯 Simple, user-friendly interface styled with vanilla CSS
